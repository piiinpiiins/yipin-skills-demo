# Yipin Skills Demo

這個資料夾包含多個實用的技能工具（Skills），主要用於影片下載、處理和轉換。

## 📋 Skills 列表

### 影片下載工具

- **[yt-video-downloader](yt-video-downloader/)** - YouTube 影片下載工具
- **[yt-mp3-downloader](yt-mp3-downloader/)** - YouTube 音訊下載工具，將影片轉換為 MP3 格式
- **[yp-get-htgaa-zoom](yp-get-htgaa-zoom/)** - HTGAA Zoom 課程錄影下載工具，支援字幕下載

### 影片處理工具

- **[yt-timelapse_4x_movie_generator](yt-timelapse_4x_movie_generator/)** - 縮時攝影產生器，快速將影片轉換成 4 倍速，保持原始比例並移除音軌

### 字幕與文字工具

- **[yt-subtitles](yt-subtitles/)** - YouTube 字幕下載與處理工具
- **[yt-transcription](yt-transcription/)** - 影片轉文字工具

### 其他工具

- **[skill-creator](skill-creator/)** - 技能建立工具，用於快速建立和驗證新的 skills
- **[skill-share](skill-share/)** - 技能分享工具
- **[postcard](postcard/)** - 明信片生成工具
- **[JOJO_style_postcard](JOJO_style_postcard/)** - JOJO 風格明信片生成工具

## 🚀 使用方法

每個 skill 資料夾內都有各自的說明文件和腳本，請進入對應資料夾查看詳細使用說明。
