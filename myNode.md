# Awesome Claude Skills (主要 28 個技能介紹)

以下是主要資料夾內各個 skill 的功能說明（以中文概述）：

## artifacts-builder
**功能**：為複雜的 claude.ai 網頁前端組件提供建立工具套件（使用 React、Tailwind CSS、shadcn/ui），適合需要狀態管理或路由的複雜介面開發。

## brand-guidelines
**功能**：將 Anthropic 官方的品牌顏色與排版風格套用到各種設計產出中，確保視覺上符合原廠設計指南。

## canvas-design
**功能**：使用設計理念建立精美的靜態視覺藝術作品（.png 與 .pdf），適合用於設計海報與插圖，並強調原創不抄襲。

## changelog-generator
**功能**：透過分析 Git Commit 歷史紀錄，將技術性的修改轉換成清楚易懂的客製化產品更新日誌（Release Notes），節省手動撰寫的時間。

## competitive-ads-extractor
**功能**：從 Facebook、LinkedIn 等廣告資料庫中萃取並分析競爭對手的廣告內容，幫助發掘有效的文案與設計創意以優化並獲取靈感。

## connect-apps / connect
**功能**：這兩個技能功能相似，旨在將 Claude 連接到外部的應用程式（如 Gmail、Slack、GitHub、Notion 及上百種第三方服務），讓 Claude 能代替你發信、建立 issue 等。

## content-research-writer
**功能**：透過自動化資料搜集、加入引用、優化文章開頭段落（Hooks）與改寫大綱，來協助撰寫高品質內容，就像有個即時回饋的協作夥伴。

## developer-growth-analysis
**功能**：分析你最近在 Claude Code 中的對話紀錄，找出寫程式的盲點與習慣，並會從 HackerNews 推薦適合的學習資源，最後給你一份量身打造的 Slack 成長報告。

## domain-name-brainstormer
**功能**：為你的新專案發想具創意的網域名稱，並自動檢查多種頂級網域（如 .com, .io, .ai 等）目前是否有人註冊，省去手動檢查的麻煩。

## file-organizer
**功能**：智慧整理電腦中的檔案與資料夾。能理解上下文、找出重複檔案、建議更佳的分類結構並自動清理，減少數位空間雜亂。

## image-enhancer
**功能**：提升圖片的解析度、銳利度與清晰度（特別是針對螢幕截圖），幫助準備高質感的簡報或社群貼文圖片。

## internal-comms
**功能**：內建公司慣用的溝通格式，幫你撰寫各種內部溝通文件（如進度報告、管理層更新、公司電子報、常見問題、事件報告等）。

## invoice-organizer
**功能**：自動幫你整理雜亂的發票與收據並分類歸檔以便報稅使用。具備讀取檔案、擷取關鍵資訊、更改一致性的檔名並分類等功能。

## langsmith-fetch
**功能**：用於除錯 LangChain 與 LangGraph 工作流（Agent）。能從 LangSmith Studio 抓取並分析對話行為、錯誤軌跡及效能表現。

## lead-research-assistant
**功能**：透過分析你的業務並搜尋目標公司，幫你找出潛在的高品質客戶（Leads）名單，同時提供相對應的聯繫策略，適合銷售與行銷人員。

## mcp-builder
**功能**：提供一份建立高品質 MCP（Model Context Protocol）伺服器的開發指南（支援 Python 與 TypeScript），讓 LLM 可以與外部 API 或工具互動。

## meeting-insights-analyzer
**功能**：分析會議記錄的逐字稿與錄音，找出你的溝通模式及行為洞察（例如：是否過度使用冗言贅字、主導發言或逃避衝突等），以提升領導溝通技巧。

## raffle-winner-picker
**功能**：從名單、試算表或 Google Sheets 中，以公平透明的隨機抽取方式完成抽獎與獲勝者挑選活動。

## skill-creator / skill-share
**功能**：提供建立與分享 Claude Skill 的能力指南。幫助使用者打造專屬新技能，並透過工具自動分享至 Slack 給團隊成員。

## slack-gif-creator
**功能**：依您的文字指示自動幫你製作專門給 Slack 頻道的 GIF 動畫（同時能判斷檔案大小限制），適合用來製作各種情境 emoji。

## tailored-resume-generator
**功能**：根據目標職缺描述（Job Descriptions），為您重新編排並產出客製化履歷，提煉出最相關的經驗與成效來最大化面試命中率。

## template-skill
**功能**：預設替換用的技能描述範本。

## theme-factory
**功能**：內建了 10 種顏色字體主題，能快速套用到您的簡報、報告與 HTML 落地頁面中，也可以隨機生出符合要求的新主題。

## twitter-algorithm-optimizer
**功能**：運用 Twitter 開源的推薦演算法邏輯來改寫與優化您的推文，確保推文格式能獲取最高推薦排名權重與觸及率。

## video-downloader
**功能**：下載 YouTube 影片並支援各式自訂畫質（1080p 至 360p）與格式選項（mp4, webm, mkv，甚至純錄音 MP3）。

## webapp-testing
**功能**：使用 Playwright 測試與互動本地的網頁應用程式，可幫助開發者驗證前端功能、除錯 UI、擷取瀏覽器截圖及檢視系統日誌。
