# System Requirements for HTGAA Zoom Recording Downloader

## Required Tools

### curl
- Version: Any recent version
- Purpose: HTTP requests and file downloads
- Installation:
  - macOS: Pre-installed
  - Linux (Ubuntu/Debian): sudo apt-get install curl
  - Linux (CentOS/RHEL): sudo yum install curl

## Optional Tools

### wget
- Version: 1.x or higher
- Purpose: Alternative download tool for large files
- Installation:
  - macOS: brew install wget
  - Linux (Ubuntu/Debian): sudo apt-get install wget
  - Linux (CentOS/RHEL): sudo yum install wget

## System Requirements

- Operating System: macOS, Linux, or WSL on Windows
- Bash: Version 4.0 or higher
- Disk Space: Varies by recording size (typically 1-5 GB per recording)
- Network: Stable internet connection for downloading large files
